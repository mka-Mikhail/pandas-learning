## О проекте

Данный проект предназначен для выполнения практических работ по дисциплине "Современные методы анализа данных и принятия решений"

## Создание виртуального окружения

Windows

```
python -m venv venv
```

Linux

```
python3 -m venv venv
```

## Активация виртуального окружения

Windows

```PowerShell
.\venv\Scripts\activate.ps1
```

```cmd
.\venv\Scripts\activate
```

Linux

```
source venv/bin/activate
```

## Установка зависимостей

```
pip install -r requirements.txt
```

## Запуск сервера Jupyter

```
jupyter notebook
```
